# VS Code Clone

A simple VS Code Clone built with React, TypeScript, Redux Toolkit, and Tailwind CSS.

## Features

- Create files
- Create folders
- Rename files and folders
- Delete files and folders
- Open multiple tabs
- Edit file content
- Syntax highlighting
- File tree explorer
- Context menu support

## Tech Stack

- React
- TypeScript
- Redux Toolkit
- Tailwind CSS
- React Syntax Highlighter
- UUID
